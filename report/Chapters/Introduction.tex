\chapter{Introduction}

This implementation of Locality Sensitive Bloom Filter and report contains several distinct parts:

\begin{itemize}
	\item A recursion based LSBF in Python
	\item A comparison between LSBF and other search methods
	\item Research that was done into implementing the project on an FPGA
	\item A critical analysis of this implementation of LSBF and improvements that can be made in future research
\end{itemize}

I hope to prove that an efficient implementation of LSBF is possible and point to research in the field of hashing algorithms that may prove useful in the investigation. \cite{Reference2}
